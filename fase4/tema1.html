Flex box

Es un método de diseño de página unidimensional para compaginar elementos en filas o columnas Los elementos de contenido se ensanchan para rellenar el espacio adicional y se encogen para caber en espacios más pequeños.miremos algunos funciones de ellos.

  • Cajas flex y inline-flex
  • Dirección: flex-direction
  • Ajuste de línea: flex-wrap
  • Dirección y ajuste: flex-flow
  • Orden: order
  • Tamaño inicial: flex-basis
  • Expansión: flex-grow
  • Compresión: flex-shrink
  • Tamaños: flex
  • Márgenes automáticos: margin
  • Alineación principal: justify-content
  • Alineación secundaria (1): align-items
  • Alineación individual: align-self
  • Alineación secundaria (n): align-content
