Listas

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número. Los tipos de listas en HTML son los siguientes:

  • Listas ordenadas . ( ol )
  • Listas desordenadas ( ul )
  • Listas de definiciones ( dl )

Enlaces

mediante los enlaces en HTML podemos comunicar una página con otra. De esta forma, enlazando documentos HTML podemos acabar tejiendo lo que es Internet. con la herramienta ( a href= ) se podra enlazar diferentes paginas. se recalca que con los enlaces se pueden tener varias utilidades como.

  • Destino del enlace: Si bien, en el enlace, podemos indicar el destino que queremos darle a dicho enlace. Eso lo podemos hacer mediante el atributo target.
  • Título de los enlaces: Podemos describir de una forma textual el destino del enlace.
  • Enlaces en HTML a una parte del documento.
  • Enlaces en HTML con imágenes
  • Enlaces en HTML para descargar fichero
